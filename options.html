Setup – inject API key from .env

The extension reads your key from .env via a one-time script. No key is ever typed into the browser UI.

  1. 1
    Make sure GEMINI_API_KEY is set in your project's .env file: 
    GEMINI_API_KEY=AIzaSy...
  2. 2
    Run the inject script from inside the chrome-extension/ folder: 
    python inject_key.py
    This writes config.js with your key.
  3. 3
    Reload the extension in Chrome:
    chrome://extensions → find Code AI → click the reload ↺ icon.
  4. 4
    Click Refresh Status above to confirm the key is loaded.

How to use

  1. 1
    Highlight any code snippet on a webpage (GitHub, Stack Overflow, MDN…)
  2. 2
    Click the Code AI toolbar icon or right-click → "Explain with Code AI ✨" then click the icon when you see the ! badge.
  3. 3
    Pick a mode: 🔍 Explain ✅ Review 🔄 Convert
  4. 4
    Click the action button and wait for Gemini's response.
